http://fr.openclassrooms.com/informatique/cours/developpez-votre-site-web-avec-le-framework-django/telechargement-et-installation-7

ensuite une fois toute est installé faite "Windows"+R, ensuite tappez la commande "cmd" et vous etes dans le shell de windows
ensuite vous vous deplacez dans le fichier ou se trouve le projet, quand vous arrivez dans le dossier ou ce trouve le script "manage.py"
vous tapez la commande suivante : "python manage.py runserver" cette commande lance le serveur et ensuite vous pouvez acceder au serveur
sur l'url de "localhost"(localhost:80/...)
